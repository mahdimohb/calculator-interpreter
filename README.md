# Calculator Interpreter

A modular arithmetic expression interpreter built using core compiler design principles.  
The project performs lexical analysis, parsing, and evaluation of arithmetic expressions with proper operator precedence and parentheses support.



## 📌 Features

- Lexical analysis (tokenization)
- Recursive descent parsing
- Operator precedence handling
- Parentheses support
- Modular architecture (Lexer, Parser, Tokens)
- Clean separation of concerns
